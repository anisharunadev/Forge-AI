## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| REST client → API handler | Untrusted input crosses here. A direct API call could otherwise bypass the approval gate. |
| LangGraph node → SDLCState | Phase transitions must be recorded in state.metadata. The decorator freezes the envelope to prevent tampering. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01-1 | Tampering | FastAPI route handler | mitigate | `@require_approval_phase(...)` decorator runs before the route body and raises `ApprovalRequiredError` if `state.pending_approval` or `state.metadata["approval:<phase>:decision"]` is missing. CI grep gate catches new handlers added without the decorator. |
| T-01-01-2 | Repudiation | Approval envelope | mitigate | `ApprovalEnvelope` is a frozen Pydantic model — once recorded in `state.metadata`, the decision cannot be mutated without leaving a forensic trace (the frozen field set prevents `model_copy(update={...})` without raising `ValidationError`). |
| T-01-01-3 | Information Disclosure | Approval decision metadata | accept | The decision metadata is already stored in `state.metadata` which is checkpointed to Postgres. No new trust boundary — the gate only reads what was already there. |
| T-01-01-4 | Elevation of Privilege | Direct REST call to phase-bound endpoint | mitigate | The decorator runs before the handler body. A direct call without a prior `ApprovalResponse` POST raises `ApprovalRequiredError` → HTTP 403. |
| T-01-01-5 | Tampering | CI grep gate | mitigate | Step 3 of `ci-hygiene-grep.yml` walks every route handler and asserts the decorator is present within 3 lines of `async def`. The allowlist comment (`# allowlist: approval-decorator`) is documented as a last-resort and must be reviewed in PR. |
| T-01-01-SC | Tampering | npm/pip/cargo installs | mitigate | slopcheck + blocking human checkpoint for any [ASSUMED]/[SUS] package; this plan adds no new third-party dependencies (decorator uses stdlib + existing Pydantic v2). |
</threat_model>
