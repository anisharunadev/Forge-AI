## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| External IdP to Forge | OIDC discovery + token exchange; client_secret encrypted at rest |
| SCIM IdP to Forge | Bearer token from Phase4ScimToken; SHA-256 hash stored |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-04-14-01 | Information Disclosure | client_secret plaintext at rest | critical | mitigate | Encrypt via core.crypto.encrypt() before INSERT |
| T-04-14-02 | Spoofing | forged SCIM token | high | mitigate | SHA-256 hash lookup; reject on mismatch |
| T-04-14-03 | Denial of Service | SCIM filter DoS | medium | mitigate | Filter grammar SUBSET (eq/ne/and/or); reject other operators with 400 |

</threat_model>
