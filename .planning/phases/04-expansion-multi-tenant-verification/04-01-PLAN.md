|----------|-------------|
| LLM client to LiteLLM cache | Cross-tenant data leak via shared cache namespace; mitigated by tenant_id prefix in key |
| Admin to invalidate endpoint | Mass-cache-flood abuse; mitigated by require_permission decorator and double-confirm on flushall |

| T-04-01-02 | Repudiation | cache bypass | medium | mitigate | PII bypass records forge.cache.bypass with pii_marked flag |
