## Decisions Made

- **PageHeader is a server component** — keeps all chrome static and SSR-friendly
- **SectionCard wraps rather than replaces shadcn Card** — preserves existing Card consumers (no breaking change)
- **EmptyState uses dashed border** — visually distinct from data-bearing sections per curated spec §6
- **Governance Center uses `<Alert variant="success|destructive">`** — semantic destructive tone over hardcoded rose/amber literals
