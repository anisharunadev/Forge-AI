<objective>
Land the application shell that the rest of Phase 0.5 (centers 0.5-04..06) plugs into: a persistent left sidebar (active-state aware, grouped, no @fora scope), a top bar with global search trigger + theme toggle, a CMD+K command palette with router + action support, page-level container with breadcrumbs, a mobile drawer (Sheet) for narrow viewports, and a refactor of app/layout.tsx to compose them. The shell is the page chrome that makes "every capability is visualized" (UI First Principle in .claude/CLAUDE.md) land consistently across all 8+ center pages.

This plan does NOT redesign any center page. The centers remain in their current state; they will plug their own headers/CTA bars into PageContainer over the next three plans.

Source spec: curated design vision (Phase 0.5 amendment, 2026-06-23) IA + chrome + Agent Status Language. Builds on the StatusPill primitive and shadcn primitives committed in 0.5-01/02.

</objective>
