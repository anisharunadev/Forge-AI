- **ApprovalTimeline destructive outline for reject** — matches Rule 3 board token semantics
- **Knowledge Center wires to typed view directly** — the existing hex-color variant in `components/knowledge-center/` is kept for now (per plan: "Do NOT touch existing bespoke views")
