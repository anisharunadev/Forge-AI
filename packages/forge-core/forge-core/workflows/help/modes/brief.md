**GSD — top commands**

```text
/forge:new-project           Initialize a project (greenfield)
/forge:map-codebase          Map an existing codebase (brownfield)
/forge:plan-phase <N>        Create a phase plan
/forge:execute-phase <N>     Execute a phase
/forge:progress              Where am I, what's next
/forge:quick                 Small ad-hoc task with GSD guarantees
/forge:fast "<task>"         Trivial inline task — no subagents
/forge:debug "<symptom>"     Persistent debug session (survives /clear)
/forge:capture               Save an idea / todo / note
/forge:ship <N>              Open a PR from a completed phase
```

More: `/forge:help` (default tour) · `/forge:help --full` (everything) · `/forge:help <topic>` (one section)
