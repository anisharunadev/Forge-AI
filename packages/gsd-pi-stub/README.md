# @opengsd/gsd-pi (stub)

Stub of the peripheral-integration layer for the GSD engine. Real package
will provide adapters for IDE integrations, git hosts, CI runners, and
chat surfaces.

For now, this stub simply re-exports the surface from `@opengsd/gsd-core`
so consumers can install a single package while the upstream split lands.

See `packages/gsd-core-stub/README.md` for the white-labeling contract.
